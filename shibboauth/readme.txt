----------------------------------------------------
Building
----------------------------------------------------
    
    Use ant to build the project:
    
        run "ant"

----------------------------------------------------
Running
----------------------------------------------------

    Using Ant (http://ant.apache.org),

        run "ant test"
    
    Otherwise, on Windows
    
        use "run.bat"
        
    Options
     -h,--help         Help output
     -l,--url          The URL to use, defaults to https://gridlab23.unibe.ch/portal/SNA/secretUserKey
     -p,--pass         Password
     -q,--proxyhost    The IP/hostname of the proxy to use
     -u,--user         Username in the form user@idphost
     -v,--verbose      Verbose logging output
     -w,--proxyport    The portname of the proxy to use    

    Be aware that the username must be in the form user@idphost. If unsure, ask the adminstrator.
